# Keboli Frontend

This is the **frontend application** built with **React, Vite, and Tailwind CSS**.

It provides a fast and modern UI with hot module replacement using Vite and utility-first styling with Tailwind.

---

## 🚀 Tech Stack

* React
* Vite
* Tailwind CSS
* TypeScript 

---

## 📦 Prerequisites

Make sure you have the following installed:

* Node.js (>= 18 recommended)
* npm or yarn

You can check with:

```bash
node -v
npm -v
```

---

## ⚙️ Installation

Clone the repository and install dependencies.

```bash
git clone https://github.com/JeneshaMalar/keboli-frontend.git
npm install
```


---

## ▶️ Running the Development Server

Start the development server:

```bash
npm run dev
```


The application will be available at:

```
http://localhost:5173
```
